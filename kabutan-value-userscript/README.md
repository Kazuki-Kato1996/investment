# Kabutan バリュー投資テーブル（Tampermonkey ユーザースクリプト）

株探の決算ページ（`https://kabutan.jp/stock/finance?code=XXXX`）を開いたときに、
「バリュー投資」シートと同じロジックの表を、タブ帯の直下に自動表示する。

## 表示される内容

ページ上部から順に:

1. **ヘッダ** - タイトル / 5年後PER 入力欄（デフォルト 18、localStorage 保存）
2. **投資判断** - 10年/5年/3年 成長率それぞれの値上がり益・配当合計・年15%/12%購入価額
3. **将来予測** - 現在〜5年後の EPS・株価（10/5/3年成長率）・PER・配当金・配当利回り
4. **基本情報** - 今期予想EPS / 今期予想配当 / 配当性向
5. **EPS成長率** - 10年/5年/3年分の CAGR・年平均成長率・採用成長率
6. **EPS履歴** - 今期予想 + 過去10年分の修正EPS・修正1株配

## 計算ロジック

### データ点と前年比

`N年分` の計算では、今期予想を先頭に `1年前`〜`N年前` までの **N+1 個** のデータ点を使う。
前年比（`今期予想/1年前`, `1年前/2年前`, ..., `(N-1)年前/N年前`）は **N 個**。

| ラベル | データ点 | 前年比 |
|---|---|---|
| 10年分 | 11点（今期予想 + 10年前まで） | 10個 |
| 5年分  | 6点  | 5個 |
| 3年分  | 4点  | 3個 |

### CAGR

```
CAGR = (今期予想EPS / N年前EPS)^(1/N) - 1
```

### 年平均成長率（分母はデータ点数 N+1）

```
年平均成長率 = Σ(前年比 - 1) / (N + 1)
```

例: 10年分の場合は `10個の前年比の合計 ÷ 11`。

### 採用成長率

```
採用成長率 = (CAGR + 年平均成長率) / 2
```

ただし **年平均成長率がマイナスの場合は CAGR を採用**（片方しか取れない場合はその値）。

### 配当・将来予測

```
配当性向      = 今期配当 / 今期EPS
n年後EPS      = 今期EPS × (1 + 採用成長率)^n
n年後配当     = n年後EPS × 配当性向
5年後株価     = 5年後EPS × 目標PER（入力値、デフォルト18）
値上がり益    = 5年後株価 - 現在株価
配当合計      = Σ(1〜5年後配当)
トータル利益  = 値上がり益 + 配当合計
年15%購入価額 = トータル利益 / 1.15^5   (= ÷2.011)
年12%購入価額 = トータル利益 / 1.12^5   (= ÷1.762)
```

※ 元のスプレッドシートでは 15% と 12% のラベルが数式と入れ違っていたため、
本スクリプトでは数学的に正しい方（15%→÷2.011、12%→÷1.762）で実装している。

### 今期予想が取れない場合

予マーク行がない、または予マーク行のEPSが欠損している場合は、
直近実績（`yearly` 末尾から遡って最初にEPSが取れる行、= 実質 1年前）を `current` として使う。

## 前提

- **株探プレミアム会員** でログイン済みのブラウザで開くこと。
  非プレミアム会員でも動くが、EPS履歴が直近5期程度しか取れず成長率の精度が落ちる。
- Chrome/Edge に Tampermonkey がインストール済み。
- **重要**: Chrome Manifest V3 では、`chrome://extensions` → Tampermonkey → 詳細 →
  「**ユーザースクリプトを許可する**」を**オン**にしないとスクリプトが注入されない。

## インストール手順

1. Chrome に [Tampermonkey](https://www.tampermonkey.net/) をインストール
2. `chrome://extensions` でデベロッパーモードを **オン**
3. Tampermonkey の「詳細」→「ユーザースクリプトを許可する」を **オン**
4. Tampermonkey ダッシュボード → 「＋」→ `kabutan-value.user.js` の中身を貼り付けて保存
5. `https://kabutan.jp/stock/finance?code=6331` などの決算ページを開く
6. タブ帯（基本情報 / チャート / 時系列 / ニュース / 決算 / 大株主）の直下に
   青枠のテーブルが表示される

## 動作の仕組み

1. ページロード後、`.kabuka` と `#stockinfo_i3 table` から現在株価・PERを取得
2. `.fin_year_result_d table` から各年度の修正EPS・修正1株配を取得
3. プレミアム向けの「過去を表示」ボタン（`#oc_b1_year_result`）を自動クリックし、
   MutationObserver で行追加を検知したら再描画
4. `#finance_box` の直前に `<div id="kk-value-table-box">` を挿入
5. ヘッダの PER 入力欄を変更すると `localStorage` に保存して `runAndRender()` を再実行

## メンテナンス

- 株探のHTML構造が変わってセレクタが効かなくなった場合は
  `scrape()` 関数内のセレクタ（`.kabuka`, `#stockinfo_i3`, `.fin_year_result_d`）を見直す。
- 計算ロジックを変えたい場合は `calc()` 関数を編集。
- ラベル・装飾を変えたい場合は `render()` 関数を編集。
- デフォルトPERは `DEFAULT_TARGET_PER` 定数で変更可能。

## 削除方法

不要になったら Tampermonkey ダッシュボードからスクリプトを削除し、
このフォルダ（`investment/kabutan-value-userscript/`）も削除すればよい。
